# Solving the Rubik's Cube 
Using OpenCV techniques in Python, colors of a **Rubik's Cube** is changed.

## How to Install
Run following command:
```
pip install -r requirements.txt
```

## How to Run
Execute this command in terminal:
```
python rubik's_cube.py
```
