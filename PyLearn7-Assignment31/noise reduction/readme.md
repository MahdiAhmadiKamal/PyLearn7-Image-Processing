
# Noise Reduction
3x3, 5x5 and 15x15 average filters are applied to reduce noise in images.

## How to Install
Run following command:
```
pip install -r requirements.txt
```

## How to Run
Execute this command in terminal:
```
python noise_reduction.py
```
