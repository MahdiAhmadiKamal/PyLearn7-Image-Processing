# Microsoft logo  
Using OpenCV techniques in Python, **Microsoft logo** is designed.

## How to Install
Run following command:
```
pip install -r requirements.txt
```

## How to Run
Execute this command in terminal:
```
python microsoft_logo.ipynb
```
