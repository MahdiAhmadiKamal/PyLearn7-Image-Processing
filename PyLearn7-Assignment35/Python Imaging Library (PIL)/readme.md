# Python Imaging Library (PIL)
Herein **OpenCV** and **pillow** libraries were utilized for image processing.

## How to Install
Run following command:
```
pip install -r requirements.txt
```

## How to Run
Execute this command in terminal:
```
python pillow.ipynb
```
